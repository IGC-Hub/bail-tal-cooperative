## 🚀 **COMMENT UTILISER LE PROJET**

### 1. **Extraire l'archive**
```bash
tar -xzf bail-tal-coop-COMPLET.tar.gz
cd bail-tal-coop
```

### 2. **Installer**
```bash
npm install
```

### 3. **Configurer Supabase**
```bash
cp .env.example .env.local
# Éditez .env.local avec vos identifiants
```

### 4. **Lancer**
```bash
npm run dev
```

### 5. **Tester!**
Ouvrez http://localhost:3000 et naviguez dans TOUTES les sections! 🎉
